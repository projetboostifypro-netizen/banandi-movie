# 🎬 Banandi Movie

Application Android de streaming de films et séries.

## Build automatique

Ce repo utilise GitHub Actions pour compiler l'APK automatiquement à chaque push.

## Comment utiliser

1. Ajoutez l'APK original dans le dossier `apk/` (nommé `original.apk`)
2. Faites un `git push` sur la branche `main`
3. GitHub Actions compile automatiquement l'APK
4. L'APK final est disponible dans les **Releases** du repo

## Télécharger

Allez dans l'onglet **Releases** pour télécharger le dernier APK compilé.
